### 字符串转换为数字的常用方法
 ```
    +'010' === 10
    Number('010') === 10
    parseInt('010', 10) === 10  // 用来转换为整数

    +'010.2' === 10.2
    Number('010.2') === 10.2
    parseInt('010.2', 10) === 10
 ```

 ### 转换为布尔型
 通过使用 **否** 操作符两次，可以把一个值转换为布尔型。
 ```
    !!'foo';   // true
    !!'';      // false
    !!'0';     // true
    !!'1';     // true
    !!'-1'     // true
    !!{};      // true
    !!true;    // true
 ```
